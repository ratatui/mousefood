# STM32 Project

## Hardware Used

1. STM32 Nucleo L433RC-P
2. DHT22 Temperature and Humidity Sensor
3. Touch Sensor
4. 0.91" (128x32) OLED display

## Connections

| Component   | Nucleo Pin/Port_Pin    |
|:--------------- | :--------------- |
| OLED   | SDA = D14/PB7, SCL = D15/PB8   |
| DHT22  | D8/PB6   |
| Touch Sensor | D4/PB5 |
